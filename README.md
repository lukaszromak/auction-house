<h1>Auction house</h1>
<hr/>
<p>Auction house CRUD application built with Spring Boot and React.</p>

<h2>Run project</h2>
<hr/>
<p>Prerequisites:</p>
<ul>
    <li>Git</li>
    <li>Docker Compose</li>
</ul>
<pre>
$ git clone https://github.com/lukaszromak/auction-house.git
$ cd auction-house
$ docker compose up

Site will be available at <a>http://localhost/</a>
</pre>
